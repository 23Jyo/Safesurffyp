About SafeSurf

Introduction

SafeSurf is a phishing domain detection tool that helps users safely analyze websites without visiting them. The tool provides a detailed analysis of URLs to assess their trustworthiness and detect potential phishing attempts.

Key Features

  • Analyze URLs for potential phishing attacks and safety risks.
  • View trust scores calculated from various security checks.
  • Preview websites without enabling external scripts for safe exploration.
  • Inspect metadata, WHOIS information, and SSL certificate details.
  • Retrieve source code of websites for detailed inspection.

How It Works

SafeSurf performs comprehensive URL assessments through multiple checks:

  • Phishtank Check: Validates URLs against the Phishtank database to detect phishing links.
  • Domain Rank: Analyzes domain popularity using the top 1 million domain list.
  • SSL Certificate: Evaluates SSL certificate details for security verification.
  • URL Analysis: Checks for URL length, depth, redirections, and IP presence.
  • Content Analysis: Scans website content for potentially malicious elements like iframes, pop-ups, and login keywords.
  • HSTS Support: Determines if the domain enforces HTTP Strict Transport Security (HSTS).

About Our Tools

  • Controller Module: The core component that orchestrates checks and calculates trust scores.
  • Model Module: Implements individual checks such as SSL validation, Phishtank lookup, and domain ranking.
  • One-Time Script: Updates the list of top domains used for domain ranking checks.
